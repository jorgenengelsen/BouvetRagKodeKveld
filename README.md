# Notebook samling for RAG kodekveld


## Oppsett


### Python / Jupyter miljø
Requirements:
- Python 3.10 (eller nyere)

Lag et venv (optional):
``` bash
python -m venv .venv
```

Installer pakker:
```bash
pip install -r requirements.txt
```


### Miljøvariabler

Kopier env template:
```bash
cp .env.template .env
```

Fyll inn nødvendige secrets i `.env`.
